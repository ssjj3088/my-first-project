# my-first-project

A simple Python utility library.

## Features

- `add(a, b)` — returns the sum of two numbers
- `greet(name)` — returns a greeting string

## Usage

```python
from utils import add, greet

print(add(2, 3))       # 5
print(greet("World"))  # Hello, World!
```

## Running Tests

```bash
python -m pytest tests/
```
